# Cowriter prompts

Use these when you want an AI (e.g. Cursor) to help write or edit a post **in your blog’s style**. The AI should use [blog-style-guide.md](./blog-style-guide.md) when helping.

---

## Start a new post with help

- *“I want to write a post about [topic]. Create the post with the script, then draft an outline (or full draft) that matches my blog style. Use the style guide in docs.”*

- *“New post: [title]. It’s a [technical / personal / devlog] post. Create the file and write a first draft following docs/blog-style-guide.md.”*

---

## Draft or expand content

- *“Help me draft the intro and TL;DR for this post. Match the voice in docs/blog-style-guide.md.”*

- *“Expand the outline in content/blog/[slug]/index.md into a full draft. Follow the blog style guide.”*

- *“Rewrite the first two paragraphs to be more conversational and hook the reader. Keep the rest of the post as-is.”*

---

## Edit for consistency

- *“Edit content/blog/[slug]/index.md so it follows docs/blog-style-guide.md (frontmatter, headings, tone, code blocks).”*

- *“Check this post against the style guide and fix anything that doesn’t match.”*

---

## Get ideas or structure

- *“I want to write about [X]. Give me 2–3 possible angles or outlines that fit my blog (use the style guide and existing posts).”*

- *“Suggest section headings for a technical post about [topic], in the style of my getting-started and case-study posts.”*

---

**Tip:** Mention “use the blog style guide” or “follow docs/blog-style-guide.md” so the AI loads the right rules. You can also invoke the **Create A Post Template** and **cowriter** skills by name if your environment supports it.
